2. Dados que Coletamos

Coletamos apenas os dados estritamente necessários para a prestação do serviço:

  • Dados de Cadastro: Nome completo, endereço de e-mail, empresa (opcional) e finalidade de uso, fornecidos voluntariamente no cadastro.

  • Consentimento (Art. 7º, I): Para envio de comunicações de marketing, quando você optar por recebê-las no cadastro.
  • Cumprimento de Obrigação Legal (Art. 7º, II): Para retenção de logs conforme exigido pelo Marco Civil da Internet.

Seus dados pessoais não são vendidos a terceiros. Podemos compartilhar dados apenas com:
